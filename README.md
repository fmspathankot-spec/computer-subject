## 🚀 Online Deploy karne ka Tarika (Step by Step)

### STEP 1 — GitHub Account banao (FREE)
1. github.com par jao
2. "Sign Up" karo — email + password
3. Account verify karo

### STEP 2 — New Repository banao
1. GitHub mein "+" button → "New repository"
2. Name likho: `computer-subject`
3. Public rakho
4. "Create repository" click karo

### STEP 3 — Files Upload karo
1. Repository mein "uploading an existing file" click karo
2. Saari files drag & drop karo
3. "Commit changes" click karo

### STEP 4 — Vercel par Deploy karo (FREE hosting)
1. vercel.com par jao
2. "Sign Up with GitHub" karo
3. "New Project" → apna `computer-subject` repo select karo
4. Framework: **Vite** select karo
5. "Deploy" click karo — BAS! 🎉

### STEP 5 — Link milega
Vercel tumhe ek FREE link dega jaise:
`https://computer-subject-xyz.vercel.app`

**Yeh link phone mein bhi kaam karega!** 📱
